# Проект "Телеграмм-бот, помощник администратора".
Данное руководство является кратким описанием основных особенностей проекта.
Проект находится в разработке, поэтому описание проекта может изменяться.

#### что представляет из себя проект?
Телеграмм-бот-помощник администратора, с графическим интерфейсом. Администратор - сотрудник, который занимается составлением графика работы для других сотрудников  организации, обработкой запросов на изготовление различных документов для сотрудников, ведением учета информации о сотрудниках. Обмен данными между администратором и сотрудниками происходит посредством мессенджера "Телеграмм"

#### для кого будет полезен данный проект?
Для организаций с ненормированным графиком работы сотрудников, в которых время смен регулярно меняется, вносятся корректировки в график работы и об этом необходимо своевременно оповещать сотрудников

#### из каких модулей состоит проект?
В проекте 6 модулей:
1. **модуль admin.pyw** - содержит графический интерфейс для администратора
2. **модуль app_logger.py** - содержит "корневой" логгер проекта, который импортируется во все остальные модули, логи записываются в файл log.txt, который находится в директории проекта.
3. **модуль botconfig.py** - содержит информацию о телеграмм-боте(имя, токен), данный модуль не выложен на github.com из соображений безопасности
4. **модуль core.py** - занимается обработкой всех вычислений, загрузкой/записью данных в базу данных
5. **модуль rabot.py** - содержит код телеграмм-бота
6. **модуль rabottest.py** - содержит тесты

вспомогательные файлы проекта:
1. **файл workers.csv** - шаблон в формате csv, используемый администратором для добавления сотрудников в базу данных
2. **файл schedule.csv** -  шаблон в формате csv, используемый администратором для добавления графика работы в базу данных

#### что может делать администратор, используя графический интерфейс проекта?
1. Добавлять/удалять новых сотрудников в базу данных
2. Добавлять/удалять график работы в базу данных
3. Осуществлять выгрузки данных о графике работы и сотрудниках из базы данных в указанные администратором файлы csv
4. Блокировать/разблокировать сотрудников(в случае увольнения/повторного трудоустройства)
5. Массово рассылать сообщения всем не заблокированным сотрудникам
6. Видеть новые запросы на выдачу копий трудовой книжки от сотрудников, помечать данные запросы как "обработанные"(сотруднику, который создал запрос приходит автоматическое оповещение от телеграмм-бота о готовности документов)

#### что может делать сотрудник, используя телеграмм-бота проекта?
1. Запрашивать свой график работы за любой месяц, имеющийся в базе данных(в результате запроса также будет отражено общее кол-во отработанных часов в месяце и зарплата)
2. Запрашивать свои персональные данные
3. Переходить по ссылке от бота на портал организации
4. Осуществлять запрос администратору на изготовление копии трудовой книжки

#### как найти бота в мессенджере телеграмм?
по имени бота: **adm_ass_bot**

#### что необходимо сделать, чтобы сотрудник получил доступ к опциям телеграмм-бота?
обратиться к администратору, чтобы администратор добавил сотрудника в базу данных,
либо разблокировал сотрудника(если сотрудник уже есть в базе данных, но был заблокирован администратором).
